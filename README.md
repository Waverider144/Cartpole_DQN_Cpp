# Cartpole_DQN_Cpp
A Cpp form of  Cartpole controled by DQN.
Note on Dependencies: This repository does not include large binary files (e.g., .dll, .lib). To run this project, you need to:

Install OpenCV 4.1.2.

Download LibTorch (CPU version) from pytorch.org.

Add the bin folders of OpenCV and LibTorch to your system's PATH or copy the missing .dll files into /Project1
